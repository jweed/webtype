ORNAMENTAL
DISPLAY
##################
Ornamental is a display serif, featuring a geometric slab serif base with variable decorative elements that shift from a sharp diamond to a rounded circle, and all shapes inbetween. the sleek “base” elements of the font allow for the ornamental elements to demand the visual attention of the reader in all uses. however, while the font utilizes many modular elemts, the strict geometry of the font allows for a swift pattern and structure in all uses. this font is ideal for large scale display text, however each letter can also be isolated to create a pattern, or used as an intricate visual in it’s own right.
